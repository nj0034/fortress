# Fortress Relay

브라우저에서 바로 열고, 초대 링크만으로 로비에 즉시 합류할 수 있는 포트리스/웜즈 스타일 탱크 대전입니다.

## 실행

```bash
npm start
```

브라우저에서 `http://localhost:3002` 을 열면 됩니다.

## 플레이 흐름

1. 호스트가 닉네임과 방 이름을 입력하고 `방 만들기`를 누릅니다.
2. 생성된 초대 링크를 복사해서 참가자에게 보냅니다.
3. 참가자는 링크를 열기만 하면 자동으로 로비 입장을 시도합니다.
4. 호스트는 로비에 모인 탱크를 확인하고 `Game Start`를 눌러 전투를 시작합니다.

이제 참가 요청 링크나 승인 링크를 다시 주고받을 필요가 없습니다.

## 조작

- `A` / `←`: 왼쪽 이동
- `D` / `→`: 오른쪽 이동
- `W` / `↑`: 포신 각도 올리기
- `S` / `↓`: 포신 각도 내리기
- `Space`: 누르고 있는 동안 파워 충전, 떼면 발사

## 탱크 종류

- `Ironclad`: 높은 체력과 넓은 폭발 범위
- `Skyrider`: 빠른 탄속과 낮은 바람 영향
- `Twin Fang`: 시간차 2연발 포격
- `Lockjaw`: 적 근처에서만 서서히 꺾이는 유도미사일
- `Aegis Nova`: 매 턴 보호막 회복

## 참고

- 게임 판정은 호스트 브라우저가 맡고, 참가자는 초대 링크만으로 직접 연결됩니다.
- 자동 입장을 위해 브라우저 피어 시그널링을 사용합니다.
- 혼자 테스트할 때는 로비에서 `봇 채우기`로 AI 탱크를 추가할 수 있습니다.
