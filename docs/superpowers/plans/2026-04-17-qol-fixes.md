## Task 1: Tank color uses tank.primaryColor in FFA, blends in team mode

**Files:** `src/render/tankRender.js`, `app.js`

- [ ] Add `resolveTankFill(match, tank, playerId, joinIndex)` in `src/render/tankRender.js`:
  ```js
  import { TANK_TYPES } from "../data/tanks.js";
  export function resolveTankFill(match, tankTypeId, playerId, joinIndex) {
    const tank = TANK_TYPES[tankTypeId] ?? Object.values(TANK_TYPES)[0];
    const tankPrimary = tank.visual?.primaryColor ?? "#ffb84f";
    const tankSecondary = tank.visual?.secondaryColor ?? "#7a4a16";
    if (match && match.teams && Object.keys(match.teams).length > 0) {
      const teamKey = match.teams[playerId];
      const teamIdx = teamKey === "red" ? 0 : teamKey === "blue" ? 1 : 0;
      const team = TEAM_COLORS[teamIdx];
      return {
        primary: mixColors(tankPrimary, team.primary, 0.4),
        secondary: mixColors(tankSecondary, team.secondary, 0.4),
      };
    }
    return { primary: tankPrimary, secondary: tankSecondary };
  }
  ```
- [ ] `renderTankToCanvas` accept `teamColor` OR `{primary, secondary}`. Keep backward compat.
- [ ] `app.js` `drawPlayer` + preview renderers: call `resolveTankFill` and pass as `teamColor: {primary, secondary}`.
- [ ] Test: FFA 4 bots render 4 different tank colors.

**Commit:** `feat(render): tank-primary color in FFA, blend with team in team mode`
